编辑器功能说明

文件▾ 菜单

导出 PDF 文件时可设置：文件名、保存目录、导出模式（单个文件或每 N 页拆分为 ZIP）。

页面▾ 菜单

调整页面尺寸（支持 A3/A4/A5/Letter/Legal），设置页面方向（纵向/横向）、边距、背景色、缩放模式。

水印▾ 菜单

添加文字或图片水印，支持位置（居中/页眉/页脚/平铺）、颜色、透明度、旋转角度设置。

印章▾ 菜单

添加电子印章图标，可设置位置和大小。

页码▾ 菜单

添加页码，设置起始页码、位置、对齐方式；支持添加 Bates 编号（含前缀、位数等）。

压缩▾ 菜单

PDF 压缩功能，支持三档级别：低压缩（高质量）、平衡模式、高压缩（小体积）。

保护▾ 菜单

设置 PDF 打开密码进行加密保护。

元数据▾ 菜单

编辑 PDF 文档信息，包括标题、作者、主题、关键词。

书签▾ 菜单

管理 PDF 书签，支持新增、编辑、删除书签，导出时自动写入 PDF。

页眉页脚▾ 菜单

添加页眉和页脚文字，支持变量替换：{{page}}、{{pages}}、{{file}}、{{date}}、{{datetime}}。

快捷操作按钮

灰度：转换为灰度模式，适合打印归档
反色：创建负片版本
扫描：模拟扫描件效果
识别：对扫描件进行 OCR 文字识别，支持中英文
归档：转换为 PDF/A 格式（1b/2b/3b）

页面操作按钮

全选/清空：选中或取消选中所有页面
预览：全屏查看当前页效果
插入：在当前选中页后插入一页空白页
前移/后移：调整页面顺序
左转/右转：旋转页面 90°
标注：直接进入当前页的画笔标注模式
裁剪：进入当前页的裁剪模式，框出保留区域即可
分割▾：将选中页面按横向或纵向切割成多份（2~10份）
恢复：撤销对当前选中页面内容（如标注、裁剪、分割）的修改，恢复到原图
删除：删除选中的页面

标注与裁剪怎么用？

1. 先在页面缩略图区选中一页，再点击工具栏的“标注”或“裁剪”。
2. 在弹窗里拖动画线即可标注；切到“裁剪”后拖出矩形框，框内区域会被保留。
3. 已有裁剪框时，可直接在框内按住拖动，微调位置，不必清空重画。
4. 点“撤销”可回退上一步；点“清除”会清空当前页全部标注和裁剪。
5. 支持 Ctrl/Cmd+Z 撤销，按 Esc 关闭弹窗。完成后直接导出或保存到空间即可生效。

常见问题

如何开始使用 Z7 PDF？

直接访问网站即可使用。点击"上传 PDF"或"开始编辑"按钮，选择本地 PDF 文件即可开始编辑。无需安装任何软件，完全在浏览器中运行。

游客和会员有什么区别？

游客可直接使用基础功能（页面整理、旋转、压缩等），每日有 3 次免费导出额度。登录成为会员后，可享受更大的文件空间、更多分享链接、在线保存文件和历史记录等功能。

编辑后的文件会自动保存在服务器吗？

游客导出默认只下载不保存；会员点击"保存到空间"后，文件才会进入个人空间，支持下次继续打开编辑。

如何分享我的 PDF 文件？

登录后，在个人空间中选中文件，点击"分享"按钮，可生成分享链接。支持设置访问密码、到期时间、下载次数限制等。

支持哪些 PDF 处理功能？

支持合并、压缩、拆分、旋转、调整尺寸、添加水印、添加页码、页眉页脚、加密解密、灰度转换、反色处理、扫描效果、PDF/A 归档、书签管理、元数据编辑等多种功能。

文件大小和数量有限制吗？

游客单文件最大 50MB，会员根据套餐不同，单文件最大可达 100-200MB。会员还有个人空间大小和文件数量限制。

页码格式有哪些？

支持多种页码格式：1（普通数字）、1/10（当前页/总页数）、Page X of Y（完整英文格式）。

页眉页脚支持哪些变量？

支持 {{page}}（当前页码）、{{pages}}（总页数）、{{file}}（文件名）、{{date}}（日期）、{{datetime}}（日期时间）等变量。

管理员如何配置站点？

管理员可进入后台管理页面（/admin.html），配置站点名称、默认套餐、注册开关、空间配额、SMTP 发信参数、支付提示等信息。
