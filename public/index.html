编辑器功能说明

文件▾ 菜单

导出 PDF 时可设置文件名、保存目录、导出模式。游客默认下载到本地；会员可继续保存到个人空间，并在下次重新打开继续编辑。

页面▾ 菜单

可统一调整页面尺寸（A3/A4/A5/Letter/Legal）、方向、边距、背景色和缩放方式。适合把不同来源的 PDF 统一成同一版式后再导出。页面设置默认不启用，重新打开文件后也会恢复为未勾选状态，避免把上一份文件的版式参数误带到新文件。

水印▾ 菜单

支持文字水印和图片水印两种模式。文字模式可设置内容、颜色、透明度、旋转角度；图片模式支持 PNG、JPG，并可设置缩放比例和位置。

印章▾ 菜单

用于在页面四角或中间叠加印章图片。支持透明度、边距、缩放和旋转，适合盖章、签章或标识“已审核”“机密”等状态。

页码▾ 菜单

支持普通页码与 Bates 编号。可设置顶部/底部、左中右对齐、字号和边距；Bates 编号支持前缀、起始值和位数控制。

压缩▾ 菜单

PDF 压缩功能，支持三档级别：低压缩（高质量）、平衡模式、高压缩（小体积）。

保护▾ 菜单

设置 PDF 打开密码进行加密保护。

元数据▾ 菜单

编辑 PDF 文档信息，包括标题、作者、主题、关键词。上传已有文档时，会自动尝试读取首个 PDF 的现有元数据，方便直接修改后导出。

书签▾ 菜单

支持一级书签的读取、编辑、删除和新增。导出时会按最终页序重新写入书签，适合整理合同汇编、教材、资料包等长文档。

页眉页脚▾ 菜单

添加页眉和页脚文字，支持变量替换：{{page}}、{{pages}}、{{file}}、{{date}}、{{datetime}}。

快捷操作按钮

灰度：转换为灰度模式，适合打印归档
反色：创建负片版本
扫描：模拟扫描件效果，并尽量保持原页面物理尺寸不变
识别：对扫描件进行 OCR 文字识别，支持中英文
归档：转换为 PDF/A 格式（1b/2b/3b）
这些开关可以叠加使用，适合批量整理打印稿、扫描件和归档文件。

页面操作按钮

全选/清空：选中或取消选中所有页面
预览：全屏查看当前页效果
插入：可插入本地 PDF、空白页，或插入左侧工作区已选 PDF
前移/后移：调整页面顺序
左转/右转：旋转页面 90°
画线/矩形/箭头/文本：直接进入当前页的可视标注模式，支持即时预览
裁剪：进入当前页的裁剪模式，框出保留区域即可
分割▾：将选中页面按横向或纵向切割成多份（2~10份）
撤销/重做：回退或恢复最近一次页面级编辑动作
删除：删除选中的页面

标注与裁剪怎么用？

1. 先在页面缩略图区选中一页，再点击工具栏的“画线”“矩形”“箭头”“文本”或“裁剪”。
2. 画线、矩形、箭头都支持直接拖动创建；文本工具单击页面后输入内容即可插入文本框。
3. 点已有标注可选中；矩形和文本框支持拖动移动、拖角点缩放，箭头支持拖动移动和端点调整，文本框双击可改内容。
4. 切到“裁剪”后拖出保留区域；已有裁剪框时，可直接在框内按住拖动微调位置。
5. 点“撤销”可回退上一步，按 Delete 可删除当前选中标注，点“清除”会清空当前页全部标注和裁剪。

支持拖拽、粘贴和多文件插入吗？

支持。你可以直接把 PDF 拖到编辑区导入，也可以在编辑器已打开文档时用 Ctrl/Cmd + V 粘贴 PDF 文件插入；通过“插入”菜单还可以把本地 PDF、空白页或左侧空间已选 PDF 追加到当前选中页后。

页面双击预览、裁剪预览和缩略图状态分别是什么？

双击缩略图会进入预览页；裁剪模式下会在页面上显示保留框，并同步展示实时裁剪预览；缩略图卡片右侧说明会显示该页是否有“裁剪”“标注”等编辑状态，方便快速检查。

页面设置、扫描、压缩这些开关会自动记住吗？

不会。页面设置、水印、印章、压缩、灰度、扫描、OCR、PDF/A、加密等会影响导出结果的选项，重新打开文件后会恢复为未启用状态。这样可以避免你在处理下一份文件时，误用上一份文件的导出参数。

“在线保存”和“导出选中 / 导出当前页”有什么区别？

“在线保存”会把当前工作台的编辑结果写回会员空间，并在保存后重新加载该文件，方便继续编辑；“导出选中 / 导出当前页”则是直接下载结果文件，不会覆盖空间中的原文件。两者都会应用当前已启用的页面设置、扫描、压缩、水印等参数。

为什么预览和最终导出有时看起来不完全一样？

工作台缩略图与全屏预览会尽量模拟最终导出效果，但页面尺寸统一、扫描、OCR、压缩等后处理最终仍以导出的真实 PDF 为准。检查 A4、横纵方向、边距或扫描效果时，建议以导出后的页面尺寸提示和下载文件为准。

长票据、超长截图和方向自动识别怎么处理？

系统会优先按页面可见内容判断方向，并在缩略图里尽量完整显示超长页面，减少因为长宽比过大导致的误判。若某些长页需要严格统一到 A4，建议启用“页面设置”，再配合“自动识别”与合适的缩放方式一起导出检查。

隐私与安全

我的 PDF 会被上传到第三方平台吗？

不会。Z7 PDF 是自部署应用，PDF 上传、编辑、导出和会员空间文件都在当前部署的服务器内处理。本版本已移除前端第三方 CDN 依赖，编辑器所需的 PDF 脚本由本服务本地提供。

游客、会员和管理员分别会保存哪些数据？

游客导出默认只下载到本机，不进入会员空间；会员主动上传、在线保存或创建分享时，文件与分享设置会写入服务器数据目录。管理员可查看站点运行所需的用户、套餐、兑换码、文件和分享记录，因此请只在可信服务器上部署并妥善保管管理员账号。

分享链接安全吗？

分享链接相当于访问凭证。公开分享知道链接即可访问；密码分享需要访问密码；登录访问要求对方登录会员账号。建议给敏感文件设置访问密码、到期时间和下载次数，文件不再需要分享时及时停用或删除分享。

部署时要注意哪些隐私安全事项？

生产环境建议启用 HTTPS，使用强管理员密码，不把 SMTP 密码、管理员密码或 Token 写进仓库；定期备份并限制 data/ 目录访问权限。若通过反向代理公开到公网，请同时配置访问日志保留策略和上传体积限制。

浏览器本地会留下文件吗？

导出的 PDF 会按浏览器下载规则保存到你的设备；浏览器也可能保留下载记录。处理敏感文件后，建议关闭不需要的分享链接，并按你的浏览器或系统策略清理下载记录和本地文件。

常见问题

如何开始使用 Z7 PDF？

直接访问网站即可使用。点击"上传 PDF"或"开始编辑"按钮，选择本地 PDF 文件即可开始编辑。无需安装任何软件，完全在浏览器中运行。

游客和会员有什么区别？

游客可直接使用基础功能（页面整理、旋转、压缩等），每日有 3 次免费导出额度。登录成为会员后，可享受更大的文件空间、更多分享链接、在线保存文件和历史记录等功能。

重新打开文件后，为什么某些勾选项被取消了？

这是为了避免误修改。像页面设置、扫描、压缩、加密、水印等都会直接影响导出结果，因此系统会在重新打开文件时将这些高风险开关重置为未启用，只保留已经保存到文件本身的页面编辑结果。

未登录可以看套餐和会员权益吗？

可以。未登录状态下进入“会员中心”，也能直接查看套餐对比和游客试用能力说明；只有在线保存、分享管理、空间文件等会员专属操作需要登录后使用。

编辑后的文件会自动保存在服务器吗？

游客导出默认只下载不保存；会员点击"保存到空间"后，文件才会进入个人空间，支持下次继续打开编辑。

左侧“我的文件”和编辑区是什么关系？

左侧“我的文件”是会员空间中的已保存文件列表，点击文件可重新打开编辑。编辑区是当前正在处理的工作台文档；通过“插入”菜单可以把空间中已选 PDF 插入到当前文档里，而不是覆盖当前编辑内容。

如何分享我的 PDF 文件？

登录后，在个人空间中选中文件，点击"分享"按钮，可生成分享链接。支持设置访问密码、到期时间、下载次数限制等。

支持哪些 PDF 处理功能？

支持合并、压缩、拆分、旋转、调整尺寸、添加水印、添加页码、页眉页脚、加密解密、灰度转换、反色处理、扫描效果、PDF/A 归档、书签管理、元数据编辑等多种功能。

文件大小和数量有限制吗？

游客单文件最大 50MB，会员根据套餐不同，单文件最大可达 100-200MB。会员还有个人空间大小和文件数量限制。

为什么有时会提示填写水印文字或上传图片？

当你开启了文字水印却没填写内容，或开启了图片水印/图片印章却没有选择图片时，系统会在导出前拦截并提示。关闭对应功能开关，或补全配置后再导出即可。

页码格式有哪些？

支持多种页码格式：1（普通数字）、1/10（当前页/总页数）、Page X of Y（完整英文格式）。

页眉页脚支持哪些变量？

支持 {{page}}（当前页码）、{{pages}}（总页数）、{{file}}（文件名）、{{date}}（日期）、{{datetime}}（日期时间）等变量。

管理员如何配置站点？

管理员可进入后台管理页面（/admin.html），配置站点名称、默认套餐、注册开关、空间配额和 SMTP 发信参数。

导出失败或页面显示异常时怎么排查？

建议先刷新页面重新导入文件，再检查是否开启了水印、印章、加密、OCR 等附加功能且缺少必要参数。如果问题只发生在某个 PDF，通常与源文件结构、扫描页或字体嵌入方式有关，换一个样本对比会更容易定位。

上传大文件、扫描件或容器重建时为什么会更慢？

首次启动、镜像重建、扫描、OCR、PDF/A 等后处理会额外消耗时间；大文件、多页扫描件和图片型 PDF 也会明显增加渲染与导出耗时。若你主要是调整页序、裁剪或简单标注，一般会比开启扫描、OCR、归档更快。
